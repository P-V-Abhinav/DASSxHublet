# Hublet Project

This is the source code directory for the Hublet real estate lead matching platform.

## Quick Start

```bash
# Make setup script executable
chmod +x setup.sh

# Run setup
./setup.sh
```

For detailed instructions, see [README.md](README.md)

## Project Structure

```
backend/          # Node.js + Express + PostgreSQL + Prisma
frontend/         # React + Vite
```
